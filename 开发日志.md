# SSH Gateway 项目开发日志

## 项目概述
SSH Gateway 是一个基于 Python 的 SSH 隧道管理平台，提供 Web UI、CLI 和 RESTful API，支持多用户认证、RBAC 权限控制和审计日志功能。

---

## 开发阶段记录

### 第一阶段：项目初始化与核心架构搭建

**日期**：项目初期
**主要工作**：
- 建立分层架构（Control layer/API, Business layer/Services, Core modules, Infrastructure）
- 实现 SSH 抽象层，支持多后端（AsyncSSH）
- 搭建 FastAPI 服务框架
- 实现 SQLite 数据库初始化

**关键文件**：
- `apps/api/main.py` - API 入口
- `modules/ssh/asyncssh_backend.py` - AsyncSSH 后端实现
- `infra/db/sqlite.py` - 数据库基础设施

---

### 第二阶段：认证系统实现

**日期**：功能开发阶段
**主要工作**：
- 实现多因素认证（MFA）框架
- 添加 bcrypt 密码加密
- 支持 TOTP 的 QR Code 生成和验证
- 集成 audit logging 到认证流程

**关键文件**：
- `apps/api/routes/auth.py` - 认证路由
- `modules/auth/mfa.py` - MFA 核心逻辑
- `web/index.html` - 认证前端界面

**API 端点**：
- `POST /auth/login` - 用户登录
- `POST /auth/mfa/setup` - MFA 设置
- `POST /auth/mfa/verify` - MFA 验证

---

### 第三阶段：隧道管理功能

**日期**：功能扩展阶段
**主要工作**：
- 实现 Local 隧道和 SOCKS5 代理隧道
- 隧道生命周期管理（创建、停止、验证）
- Web UI 隧道状态实时展示
- 终端按钮功能，跳转到命令输入页面

**关键文件**：
- `apps/api/routes/tunnel.py` - 隧道路由
- `services/tunnel_service.py` - 隧道业务逻辑
- `modules/tunnel/manager.py` - 隧道管理器
- `web/terminal.html` - Web 终端页面

**API 端点**：
- `GET /tunnel/list` - 列出用户隧道
- `POST /tunnel/create` - 创建隧道
- `POST /tunnel/stop/{tunnel_id}` - 停止隧道
- `GET /tunnel/verify/{tunnel_id}` - 验证隧道
- `POST /tunnel/exec/{tunnel_id}` - 执行命令

---

### 第四阶段：权限系统扩展（ACL → RBAC）

**日期**：安全增强阶段
**主要工作**：
- 从 ACL 扩展到基于角色的访问控制（RBAC）
- 新增 `roles` 和 `user_roles` 数据库表
- 实现策略评估器 `evaluator.py`
- 集成 X-User Header 进行身份传播

**关键文件**：
- `infra/db/sqlite.py` - 数据库表结构更新
- `modules/auth/evaluator.py` - 策略评估器
- `apps/api/routes/auth.py` - RBAC 端点

**数据库变更**：
```sql
CREATE TABLE roles (
    name TEXT PRIMARY KEY,
    allow_patterns TEXT,
    deny_patterns TEXT
);

CREATE TABLE user_roles (
    username TEXT,
    role_name TEXT,
    PRIMARY KEY (username, role_name)
);
```

---

### 第五阶段：CLI 工具完善

**日期**：用户体验优化阶段
**主要工作**：
- 使用 Rich 库优化输出格式
- 添加 X-User Header 支持
- 实现隧道验证命令
- 改进错误处理和提示信息

**关键文件**：
- `apps/cli/main.py` - CLI 主入口

**可用命令**：
- `ssh-gateway login <username>` - 登录
- `ssh-gateway tunnel list` - 列出隧道
- `ssh-gateway tunnel create` - 创建隧道
- `ssh-gateway tunnel stop <tunnel_id>` - 停止隧道
- `ssh-gateway tunnel verify <tunnel_id>` - 验证隧道

## 技术栈

| 分类 | 技术/库 |
|------|---------|
| Web 框架 | FastAPI |
| 数据库 | SQLite (aiosqlite) |
| SSH | AsyncSSH |
| HTTP 客户端 | httpx |
| 密码学 | bcrypt, pyotp |
| CLI 美化 | Rich |
| 数据验证 | Pydantic |
| 前端 | 原生 JavaScript, HTML, CSS |
